Описание библиотеки

Данная страница посвящена описанию библиотеки для анализа текстур TexturalAnalysis.
Библиотека решает следующие задачи:

  • анализ текстур на основе текстурных признаков
  • R/S - анализ
  • анализ текстур на основе расчета фрактальной размерности
Программно текстура задается в виде матрицы. значениями которой являются интенсивности пикселей в интервале от 0 до 255, для черно-белого изображения.

Инструкция

Анализ текстур на основе текстурных признаков

Для следуюших признаков на вход необходимо передать матрицу, соответствующую изображению, на выходе получается новая матрица, соответствующая результату применения выбранного признака. Для этого необходимо подключить один из файлов, соответствующих признаку и вызвать соответсвующую функцию.
  • Признаки, использующие матрицу смежности значений яркости:
    • энергия “Energy.h”
    • энтропия “Entropy.h”
    • максимальная вероятность “MaxProbability.h”
    • контраст “Contrast.h”
    • обратный момент “Moment.h”
  • Признаки, описывающие текстуры длинами серий
    • обратные моменты увеличены при коротких сериях “Haralic1.h”
    • моменты увеличены при длинных сериях “Haralic2.h”
    • неоднородность яркости “Haralic3.h”
    • неоднородность длины серий “Haralic4.h”
    • доля изображения в сериях “Haralic5.h”

R/S - анализ

Для применения R/S-анализа, необходимо подключить файл “RS.h” и воспользоваться функцией calculateHirstCoefficient(Image, x1, y1, x2, y2), которая возвращает коэффициент Хёрста; где x1, y1, x2, y2 - координаты точек начала и конца отрезка, на котором необходмо провести вычисления. Для того, чтобы получить точки для построения графиков, необходимо воспользоваться функцией calculatePoints(Image, x1, y1, x2, y2).
